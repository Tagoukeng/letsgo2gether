wir brauchen jquery und css und js um Bootstrap lokal auszuführen.
Ein Raster-konzept vereinfacht uns das Leben in Bootstrap indem wir unsere Seiten responsiv und flexible gestalten können.
Wir unterscheiden container , row und col
Beispiel : 
<div class="container">
    <div class = "row">
        <div class = "col-md-4">Eine von 3 Spalten </div>
        <div class = "col-md-4">Die 2. von 3 Spalten </div>
        <div class = "col-md-4">Die 3. von 3 Spalten </div>
    </div>
</div>
Es gibt 2 Arten von container : container-fluid hat eine flexible Größe und container hat fixe Größe mit definierten "Break points" , es rastet also jedesmal wenn einen Break-point erreich wird.
wir unterscheiden : col-xs , col-sm , col-md , col-lg , col-xl. Man kann sie gut kombinieren ein Bsp habe ich in Index.
